# 参考资料 (References)

这个目录存放当前使用的Excel表格和其他参考文档，用于指导 Atlas 系统的设计和开发。

## 目录说明

将现有的业务表格、流程文档等参考资料放在这里，包括但不限于：

- 资产管理相关的Excel表格
- 采购流程相关文档
- 库存管理表格
- DCIM相关资料
- 审批流程模板
- 其他业务参考文档

## 使用说明

这些文件仅作为需求分析和系统设计的参考，帮助理解：
- 现有的业务流程
- 数据字段和结构
- 业务规则和约束
- 用户使用习惯

在设计数据模型和API时，会参考这些文件来确保系统满足实际业务需求。

## 注意事项

- 请确保上传的文件不包含敏感信息（如真实的序列号、价格等）
- 如果包含敏感信息，请先脱敏处理
- 文件命名建议使用有意义的名称，如：`资产入库表.xlsx`、`采购订单模板.xlsx` 等
